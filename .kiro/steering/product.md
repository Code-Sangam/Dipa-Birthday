# Product Overview

This is a birthday surprise website - an interactive, animated web experience designed to celebrate someone's special day. The application presents a multi-screen journey with:

- Countdown loader with animated spinner
- Personalized intro screen with birthday message
- Interactive cake screen with candle animation
- Balloon popping screen
- Photo gallery slideshow
- Final heartfelt message screen

The project is built as a customizable template where users can personalize messages, photos, and content for their loved ones. It emphasizes smooth animations, emotional design, and a delightful user experience.
