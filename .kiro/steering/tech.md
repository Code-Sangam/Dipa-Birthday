# Tech Stack & Build System

## Framework & Core Libraries
- **Next.js 15.5.4** - React framework with App Router
- **React 19.1.0** - UI library
- **Tailwind CSS 4** - Utility-first CSS framework
- **Framer Motion 12** - Animation library for smooth transitions and effects
- **Kalam font** (Google Fonts) - Primary typeface for handwritten feel

## Key Dependencies
- `canvas-confetti` - Confetti effects for celebrations
- `lucide-react` - Icon library
- `swiper` - Touch slider for photo gallery

## Development Tools
- ESLint - Code linting
- PostCSS - CSS processing

## Common Commands
```bash
# Development server (runs on http://localhost:3000)
npm run dev

# Production build
npm run build

# Start production server
npm start

# Run linter
npm run lint

# Install dependencies
npm install
```

## Build Configuration
- Uses default Next.js configuration (minimal `next.config.mjs`)
- Tailwind CSS configured via PostCSS
- ESLint with Next.js recommended rules
